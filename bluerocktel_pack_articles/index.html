Articles

Cette page vous indique comment compléter l'onglet "Articles" de votre pack BlueRockTEL. En savoir plus sur le pack BlueRockTEL.

Actif

active, boolean default true

Cette colonne vous permet d'indiquer si l'article est actif ou non.

Indiquer 1 pour actif, 0 pour inactif.

Actif pour la facturation ponctuelle

activeForInitialBilling

Cette colonne vous permet d'indiquer si l'article est actif pour la facturation ponctuelle ou non.

Indiquer 1 pour actif, 0 pour inactif.

Actif pour la facturation récurrente

activeForRecurringBilling

Cette colonne vous permet d'indiquer si l'article est actif pour la facturation récurrente ou non.

Indiquer 1 pour actif, 0 pour inactif.

Actif pour la facturation des licences

actifForLicences

Cette colonne vous permet d'indiquer si l'article est actif pour la facturation des licences ou non.

Indiquer 1 pour actif, 0 pour inactif.

Fréquence de facturation

billingFrequency

Cette colonne vous permet d'indiquer la fréquence de facturation standard de l'article.

Référence interne

internalReference

Désignation

description, string

Cette colonne vous permet d'indiquer la désignation de l'article.

Cette colonne vous permet d'indiquer une troisième ligne de désignation de l'article, le cas échéant.

Cette colonne vous permet d'indiquer une cinquième ligne de désignation de l'article, le cas échéant.

Désignation ligne 6

description6

Cette colonne vous permet d'indiquer une sixième ligne de désignation de l'article, le cas échéant.

Désignation ligne 7

description7

Cette colonne vous permet d'indiquer une septième ligne de désignation de l'article, le cas échéant.

Désignation ligne 8

description8

Cette colonne vous permet d'indiquer une huitième ligne de désignation de l'article, le cas échéant.

Désignation ligne 9

Famille

family_id, integer

Cette colonne vous permet d'indiquer l'id de la famille d'articles du produit.

Sous-famille

subfamily_id, integer

Prix d'achat hors taxe

buyingPriceWithoutTax

Cette colonne vous permet d'indiquer le prix d'achat hors taxe de l'article. Vous pouvez préciser jusqu'à 4 décimales.

Possibilité de modifier le prix d'achat hors taxe pour l'article

buyingPriceWithoutTaxIsChangeable

Prix d'achat hors taxe des frais d'accès au service

buyingServiceAccessFeesWithoutTax

Cette colonne vous permet d'indiquer le prix d'achat hors taxe des frais d'accès au service. Vous pouvez préciser jusqu'à 4 décimales.

Prix de vente hors taxe

sellingPriceWithoutTax

Cette colonne vous permet d'indiquer le prix de vente hors taxe de l'article. Vous pouvez indiquer jusqu'à 4 décimales.

Prix de vente hors taxe des frais d'accès au service

sellingServiceAccessFeesWithoutTax

Cette colonne vous permet d'indiquer le prix de vente hors taxe des frais d'accès au service. Vous pouvez indiquer jusqu'à 4 décimales.
