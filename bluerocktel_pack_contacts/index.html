Contacts

Cette page vous indique comment compléter l'onglet "Contacts" de votre pack BlueRockTEL. En savoir plus sur le pack BlueRockTEL.

Client

contactable_type, string, 'App\Customer' or 'App\Prospect'

contactable_id, integer

Cette colonne vous permet d'indiquer à quel client le contact est rattaché.

Nous vous invitons à indiquer dans cette colonne le numéro de la ligne correspondant au client dans l'onglet "Clients". À défaut, vous pouvez indiquer le compte client.

Civilité

civility, integer

Cette colonne vous permet d'indiquer la civilité du contact, sous forme numérique, en respectant la table de correspondance ci-après :

  • 1 pour Mademoiselle
  • 2 pour Madame
  • 3 pour Monsieur

Prénom

firstName, string

Cette colonne vous permet d'indiquer le prénom du contact.

Nom

lastName, string

Cette colonne vous permet d'indiquer le nom du contact.

Adresse email

emailAddress, string

Cette colonne vous permet d'indiquer l'adresse email du contact.

Téléphone mobile

mobilePhone, string

Cette colonne vous permet d'indiquer le téléphone mobile du contact.

Téléphone fixe

landlinePhone, string

Cette colonne vous permet d'indiquer le téléphone fixe du contact.

Rôle

role

Cette colonne...

Service

service

Cette colonne...

Actif

active, boolean, default true

Cette colonne vous permet de préciser si le contact est actif ou non.

Indiquer 1 pour actif ou 0 pour inactif.

Représentant légal

legalRepresentative

Cette colonne vous permet de préciser si le contact est un représentant légal de la société ou non.

Indiquer 1 pour représentant légal ou 0 dans le cas contraire.

Destinataire des factures

invoicesRecipient, boolean default true

Cette colonne vous permet d'indiquer si le contact recevra les factures.

Indiquez 1 s'il doit recevoir les factures et 0 dans le cas contraire.

Destinataire des newsletters

newslettersRecipient

Cette colonne vous permet d'indiquer si le contact recevra les newsletters.

Indiquez 1 s'il doit recevoir les newsletters et 0 dans le cas contraire.

Accès à l'espace clients

myClient, boolean, default true

Cette colonne vous peremt d'indiquer si le contact disposera d'un accès à l'espace clients ou non.

Espace client : admin (factures, relevés, documents)

myClientAdmin, boolean, default true

Cette colonne vous permet d'indiquer si le contact, une fois connecté à l'espace clients, pourra accéder aux factures, aux relevés de consommations et aux documents administratifs partagés.

Indiquer 1 ou vrai ou 0 pour faux.

Espace client : services avancés (configuration des services Telecom)

myClientAdvancedServices

L'espace clients BlueRockTEL peut être configuré de telle sorte qu'il donne au client la possibilité de configurer ses services Telecom : par exemple modifier les tables de routages horaires, géographiques ou intelligents pour les appels entrants.

Cette colonne vous permet d'indiquer si le contact, une fois connecté à l'espace clients, pourra accéder à la configuration des services Telecom du client.

Indiquer 1 pour vrai ou 0 pour faux.

Espace client : service SMS

L'espace clients BlueRockTEL peut être configuré de telle sorte qu'il donne au client la possibilité d'envoyer des SMS.

Cette colonne vous permet d'indiquer si le contact, une fois connecté à l'espace clients, pourra accéder au service d'envoi de SMS.

Indiquer 1 pour vrai ou 0 pour faux.

Contact technique

BlueRockTEL vous permet d'indiquer si le contact est contact technique ou non, notamment pour partager avec lui certaines informations et alertes.

Dans l'espace clients, le contact technique accèdera au ticketing (support).

Indiquer 1 pour vrai ou 0 pour faux.

Accès API

BlueRockTEL vous permet de donner à certains contacts un accès API à l'espace clients.

Une bonne pratique est souvent de ne pas donner un accès API aux contacts réels de l'entreprise, c'est-à-dire les utilisateurs de votre espace clients, mais de créer un contact spécifique pour l'accès API.

Indiquer 1 pour vrai ou 0 pour faux.

Commentaire

comment, text

Cette colonne vous permet d'ajouter un commentaire sur le contact.
