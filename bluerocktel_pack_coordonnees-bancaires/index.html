Coordonnées bancaires

En savoir plus sur le pack BlueRockTEL

Client

customer_id, integer

Cette colonne vous permet d'indiquer à quel client les coordonnées bancaires sont rattachées.

Nous vous invitons à indiquer dans cette colonne la raison sociale du client, orthographiée de la même façon qu'elle l'est dans la table des clients.

IBAN

IBAN, string

Cette colonne vous permet d'indiquer l'IBAN du client.

BIC

BIC, string

Cette colonne vous permet d'indiquer le code BIC associé à l'IBAN du client.

id customer_id IBAN BIC mandateIdentification DtOfSgntr numberOfDebit
