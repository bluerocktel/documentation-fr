Coordonnées bancaires

Cette page vous indique comment compléter l'onglet "Coordonnées bancaires" de votre pack BlueRockTEL. En savoir plus sur le pack BlueRockTEL

Client

fileable_type, string, 'App\Customer'

fileable_id, integer

Cette colonne vous permet d'indiquer à quel client les coordonnées bancaires sont rattachées.

Nous vous invitons à indiquer dans cette colonne la raison sociale du client, orthographiée de la même façon qu'elle l'est dans la table des clients.

IBAN

IBAN, string

Cette colonne vous permet d'indiquer l'IBAN du client. C'est une information obligatoire pour que BlueRockTEL puisse produire le fichier de prélèvements SEPA.

BIC

BIC, string

Cette colonne vous permet d'indiquer le code BIC associé à l'IBAN du client. C'est une information obligatoire pour que BlueRockTEL puisse produire le fichier de prélèvements SEPA.

Mandat de prélèvement SEPA

mandateIdentification, string

Cette colonne vous permet d'indiquer la référence unique de mandat du client. C'est une information obligatoire pour que BlueRockTEL puisse produire le fichier de prélèvements SEPA.

Date de signature du mandat

DateOfSgntr, string

Cette colonne vous permet d'indiquer la date de signature du mandat. C'est une information obligatoire pour que BlueRockTEL puisse produire le fichier de prélèvements SEPA.

Nombre de prélèvements

numberOfDebit, integer

Cette colonne vous permet d'indiquer le nombre de prélèvements opérés sur ces coordonnées bancaires dans le passé.
