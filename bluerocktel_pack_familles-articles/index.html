Familles d'articles

Cette page vous indique comment compléter l'onglet "Familles d'articles" de votre pack BlueRockTEL. En savoir plus sur le pack BlueRockTEL.

BlueRockTEL vous permet d'organiser les articles de votre catalogue en familles d'articles, puis en sous-familles.

Code

code, string

Cette colonne vous permet d'identifier votre famille par un code.

Nom

name, string

Cette colonne contient le nom de la famille.

Compte de produit

revenueAccount, string

Cette colonne contient le compte de produit correspondant à la famille d'articles.

Cette information est utile notamment pour les exports vers la comptabilité.

Compte de charge

expenseAccount, string

Cette colonne contient le compte de charge correspondant à la famille d'articles.

Cette information est utile notamment pour les exports vers la comptabilité.

Chiffre d'affaires

turnover, boolean, default true

Il peut arriver que vous soyez amené à facturer certains articles qui n'entrent pas dans votre chiffre d'affaires : c'est notamment le cas des dépôts de garantie.

Cette colonne vous permet d'indiquer si la famille d'articles doit entrer dans le chiffre d'affaires ou non.

Indiquer 1 si la famille doit entrer dans le chiffre d'affaires, ou 0 dans le cas contraire.

Famille concernée par l'auto-liquidation de TVA (opérateurs ARCEP)

VATAutoReverseOperators

Indiquer 1 si vous souhaitez appliquer le mécanisme de TVA auto-liquidée pour cette famille, 0 dans le cas contraire.

Appliquer automatiquement les frais de livraison pour cette famille

autoApplyDeliveryFees, boolean, default false

Vous pouvez configurer BlueRockTEL pour appliquer automatiquement des frais de livraison lors de la création des devis, contrats puis factures.

Cette colonne vous permet d'indiquer si la famille d'articles est concernée ou non par l'application automatique des frais de livraison.

Indiquer 1 pour appliquer automatiquement les frais de livraison, 0 dans le cas contraire.

Appliquer automatiquement les frais d'exploitation pour cette famille (indiquer le taux)

Vous pouvez configurer BlueRockTEL pour délivrer automatiquement des rapports de rémunération pour vos commerciaux, que ceux-ci soient salariés ou indépendants, en combinaison avec un apporteur d'affaires (rémunération partagée) ou non.

La rémunération commerciale peut être calculée soit sur le chiffre d'affaires, soit sur la marge.

Dans le cas d'une rémunération commerciale calculée sur la marge, il peut être intéressant de prévoir des frais d'exploitation : en effet, si la marge brute est le différentiel entre votre prix d'achat et votre prix de vente, votre entreprise supporte d'autres frais (facturation, support...)

Les frais d'exploitation pouvant être très variables d'une famille à une autre, BlueRockTEL vous permet d'indiquer un taux différent pour chaque famille d'articles.

Cette colonne vous permet d'indiquer le taux de frais d'exploitation applicable à la famille d'articles.

Catalogue

catalog, boolean, default true

Cette colonne vous permet d'indiquer si la famille d'articles soit apparaître dans le catalogue.

Indiquer 1 ou vrai, ou 0 pour faux.
