Mandats GoCardless

Cette page vous indique comment compléter l'onglet "GoCardless" de votre pack BlueRockTEL. En savoir plus sur le pack BlueRockTEL.

Si vous n'utilisez pas le service GoCardless pour prélever vos clients, vous pouvez bien entendu ignorer cet onglet.

Customer

customer_id, integer

Cette colonne vous permet d'indiquer à quel client le contact est rattaché.

Nous vous invitons à indiquer dans cette colonne le numéro de la ligne correspondant au client dans l'onglet "Clients". À défaut, vous pouvez indiquer le compte client.

Mandat

mandate, string

Indiquer dans cette colonne la référence du mandat GoCardless de votre client.

Cette valeur commence par les caractères "MD".

Customer

customer, string

Indiquer dans cette colonne la référence client GoCardless de votre client.

Cette valuer commence par les caractères "CU".

Email

email, string

Indiquer dans cette colonne l'email associé à votre client dans l'interface GoCardless.

Information libre

custom, string

Vous pouvez indiquer, dans cette colonne, la valeur "Custom" associée à votre client dans l'interface GoCardless.
